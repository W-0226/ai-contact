# 测试用例 2 — 赵总（信息稀疏型 / 僵尸人脉）

> **输入原文**：
> 赵总，上次在行业峰会认识的，做AI芯片的，
> 加了微信但一直没联系。名片上写着他是某公司CTO。
> 想约他聊合作但不知道从什么话题切入。

### 能力边界分析

这是典型的「加了微信但从未聊过」的僵尸人脉：
- 本地规则引擎只能提取"合作"这个关键词事件，无法理解"从未联系"
- 「认识途径」「行业」「破冰建议」需 LLM 语义理解
- 置信度 0.35 < 0.6，会触发 hybrid 模式的 LLM 增强
